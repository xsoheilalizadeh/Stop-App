## Stop-App 

Stop-App is a PowerShell module for stopping apps, fast and easily.

### Installation
Run follwing command in PowerShell:

```ps1
Install-Module -Name Stop-App -Force -AcceptLicense
```

### Commands

```ps1
Stop-Chrome
Stop-VSCode
Stop-OneDrive
.
.
.
```
